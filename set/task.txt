Сегодня Вам предстоит самостоятельно релизовать интерфейс Set (на базе HashMap). Как не трудно догдаться имя реализации интферфейса Set: HashSet

Напомним Вам, что интерфейс Set полностью дублирует методы интерфейса Collection, с тем лишь исключением, что он добавляет новые ограничения на методы.

Ваш интерфейс должен соответствовать описанию в официальной документации.
